5.4.2 Dynamic aggregation

If no column is specified in highlight_key(), it uses the row index to inform the SQL query, which is often desirable for scatterplot brushing.

d <- highlight_key(mpg)
base <- plot_ly(d, colors = "Set1", color = ~class)

base %>% 
  add_markers(x = ~displ, y = ~jitter(cyl)) %>%
  subplot(base %>% add_boxplot(x = ~class, y = ~cty)) %>%
  highlight("plotly_selected", opacityDim = 0.6)
